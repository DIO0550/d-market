# Review TypeScript Naming

## Description

TypeScriptプロジェクトで命名規則への準拠をレビューし、一貫性のある可読性の高いコードを実現するコマンドです。

## Prompt Template

`typescript-naming-reviewer`エージェントを使用して、コードの命名規則をレビューしてください。

以下のタスクを実行してください：

1. **スキルの参照ファイルを使用して命名規則を取得する**

   - `typescript-code-review-skill:naming-review` を取得し、レビュー基準を確認

2. **コード内の識別子を分析する**

   - 変数（const、let、var）
   - 関数とメソッド
   - クラスとインターフェース
   - 型エイリアスとEnum
   - プロパティとパラメータ
   - ファイルとモジュール名

3. **命名規則への準拠をチェックする**

   - camelCase: 変数、関数、メソッド名
   - PascalCase: クラス、インターフェース、型、Enum
   - UPPER_SNAKE_CASE: 定数
   - 説明的で意味のある名前か
   - 略語や単一文字変数の回避

4. **改善提案を提供する**
   - 違反している各名前に対して推奨される修正を提示
   - 影響レベル（重大、中程度、軽微）を分類
   - 全体的な準拠スコアを算出

## Notes

- 最近追加または変更されたコードに焦点を当てる
- プロジェクト固有の例外を考慮
- サードパーティコードや生成されたファイルは除外
- レビュー結果は日本語で提供
